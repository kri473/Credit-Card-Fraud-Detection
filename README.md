# Credit-Card-Fraud-Detection

## Overview
This project uses Machine Learning to detect fraudulent credit card transactions. The model learns transaction patterns and identifies suspicious activities with high accuracy.

## Dataset
- Credit Card Transactions Dataset
- Binary Classification:
  - 0 → Legitimate Transaction
  - 1 → Fraudulent Transaction

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Handling Class Imbalance
5. Model Training
6. Model Evaluation
7. Fraud Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results
The trained model effectively identifies fraudulent transactions while minimizing false positives.
